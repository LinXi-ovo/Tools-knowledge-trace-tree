# 知识树管理系统用户手册

## 菜单功能

1. **初始化数据库** – 创建表并插入示例知识树。
2. **添加新节点** – 输入名称、层级、类别、父节点ID。
3. **删除节点** – 按ID删除，同时删除所有子节点。
4. **修改节点** – 修改名称、类别。
5. **按ID查看** – 查看指定ID的节点信息。
6. **关键词搜索** – 模糊搜索节点名称。
7. **查看子树** – 显示某节点及其全部后代。
8. **统计类别** – 统计各类型节点数量，显示最大层级。
9. **显示根节点子节点** – 快速查看一级分类。

## JSON 导入

主程序菜单中有独立的“导入JSON”选项，使用 `import_tree.json` 格式：
```json
{
  "name": "根节点",
  "level": 0,
  "type": "类别",
  "children": [ ... ]
}
```
导入会自动适应数据库现有列，缺失字段用 NULL 填充。

## 数据库迁移

当需要新增字段时：
1. 修改 `migrate_db.py`，在列表中添加 `(列名, 类型)` 元组。
2. 运行 `python migrate_db.py`。
3. JSON 文件中添加对应字段即可。

## 注意事项

- 删除节点不可逆，请谨慎操作。
- 表结构变更前建议备份 `knowledge_tree.db`。
